noteRanges :: Note Symbol Ann -> [(Range, [(Text, Range)])] Source #

Returns a list of ranges where this note should be marked in the document, as well as a list of related ranges the note might refer to, and their relevance.

E.g. a name conflict note might mark each conflicted name, and contain references to the other conflicted name locations.
